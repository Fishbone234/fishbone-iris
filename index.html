Iris Biometric Authentication System

This prototype demonstrates an iris-based biometric system including user registration and identity verification.

1. User Registration

The user registers their iris pattern into the system.

Align your eye with the scanner

2. Identity Verification

The system scans the iris and compares it with the registered template.

System Overview

Iris recognition is a biometric technology that identifies individuals by analyzing unique patterns in the iris. This prototype simulates the registration and verification process to demonstrate usability and system flow.

* This is a simulated biometric system for educational purposes only. No real biometric data is collected or stored.
